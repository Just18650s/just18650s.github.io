why bro bro?

Pack your recycled 18650 Li-ion cells into optimal packs for your bike, power wall or other 18650 projects.

We would love to hear what need improving...

Thank you to everyone for your feedback, every change implemented to date is because of one of your requests. We take the most requested features and try implement that as quickly as possible.

If you have an idea or an issue, tell us in as much detail as you can. If it's an issue then sample inputs would really help us to fix them.

News

17/04/2020

Because some people liked the export to CSV, and some people liked to copy/paste the results. I've added an extra button "Copy to Clipboard".

16/04/2020

Updated list of requested features, if you want one of these let us know...

  • Put the dimensions screen before the cells screen.
  • Allow copy and paste of pack information rather than clicking export
  • Include internal resistance as a factor when packing cells
  • Allow an ID to be specified for easier identification of a cell
  • Allow for cells to be maintained and stored by the website
  • Save/edit generated packs
  • Handle cells that are left in pairs in packing

Currently internal resistance is quite highly requested. So any suggestion on how that could be specified along side a capacity would be helpful.

16/04/2020

Bit of a gap between updates due to another round of complex changes under the hood. The important part is what's new...

On the built pack screen there is now a toggle for Compact layout which brings back the layout from the older version as some people found that useful when they are physically building their packs.

There was a bug in the variable packing that has finally been tracked down, fixing the bug means it's now able to do a better job than before, nothing major.

And finally, you can now specify the nominal voltage of your cells. This only has the effect of showing the right voltage on the pack summary on the built pack screen.

08/04/2020

Few labeling updates for the built pack screens to try make it clearer what is in parallel and in serial.

07/04/2020

Few small changes today, stats were showing a higher number of mobile phone users that expected so layout has been optimized a little for those people.

Finally at the end of the built packs screen it now shows you cells that were not used, and the FAQ explains why this happens.

06/04/2020

Feedback has spoken the export to CSV functionality was the most important feature to happen next. It's now available as the "Export CSV" button on the built packs screen.

03/04/2020

Recently I've had a bug report or two in feedback, which is very welcome! For me to be able to work with you to fix them please raise any issues here.

02/04/2020

As promised marking cells that are used/found on a built pack is now available again. Some updates to the FAQ's added thanks to feedback and questions left.

I've distilled all the feedback into the following items that need to be worked on next, let me know which you think should be the next thing to work on...

  • Include internal resistance as a factor when packing cells
  • Allow an ID to be specified for easier identification of a cell
  • Allow for cells to be maintained and stored by the website
  • Save generated packs for later
  • Edit saved packs
  • Show cells not used after packing
  • Export to CSV / Excel
  • Handle cells that are left in pairs in packing

31/03/2020

First release with some changes above and beyond the old version. We now have a less bland layout, and my shocking attempt at graphic design for a logo. There is also the addition of a FAQ page, any questions i get in the feedback i'll answer the best i can here.

Most importantly the pack builder now lets you specify a capacity for your pack rather than just giving a number of cells to arrange in parallel. This will still result in a pack with a fixed number of cells this just makes it easier to specify the capacity rather than working it out yourself. The last new feature is generated packs now show pack statistics (voltage, Ah and Wh).

There used to be a feature that let you check off used cells, that will come back soon, oversight on my part while reworking the site. This page will also work as bit of a change log, I'll add things here as I update features.
